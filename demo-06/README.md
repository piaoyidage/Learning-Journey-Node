# Node 初级入门到高级入门实战

## 目标

完成一个完整的 web 应用，该应用允许用户浏览页面以及上传文件。

1. 用户可以通过浏览器使用我们的应用。
2. 当用户请求 `http://domain/start` 时，可以看到一个欢迎页面，页面上有一个文件上传的表单。
3. 用户可以选择一个图片并提交表单，随后文件将被上传到 `http://domain/upload`，该页面完成上传后会把图片显示在页面上。


## 知识点

1. 事件驱动
2. 模块
3. 路由选择
